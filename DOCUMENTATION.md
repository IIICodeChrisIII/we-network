# WE Network — User Manual & Features

Welcome to the WE Network! This documentation describes the available features and how to navigate the platform.

## Overview

The WE Network is an internal social network that directly connects students with experts and HR personnel from Würth Elektronik. Users can exchange ideas, find jobs, and create their own talent profile for potential employers.

## Roles in the System

There are three main user groups, each with access to different features:
- **Students:** Can network, chat, and create a profile for the talent pool.
- **Specialists (Würth Employees):** Identifiable by a red badge. Available to answer technical and professional questions.
- **Admin / HR:** Have access to the Analytics Dashboard, can create new chat groups, and search the talent pool for candidates.

## Navigation & Main Features

The main navigation is located at the top of the screen (or in a sidebar) and provides access to the following areas:

### 1. Community Feed (News)
- **Function:** The homepage and central hub for updates.
- **Features:** 
  - Read posts from other users.
  - Post your own news or questions.
  - Comment on and "like" posts.
  - Solve special tech challenges to collect "WE-Nodes" (reward points).

### 2. Live Chat (Channels)
- **Function:** Group chats, organized by topics or universities.
- **Features:** 
  - Join public channels or specific university groups.
  - Send and receive messages in real-time.
  - Click on a username (Hover-Card) to view their profile or start a direct message.
  - *Admins* can create new channels here.

### 3. Messages (DMs)
- **Function:** Private one-on-one chats with selected contacts.
- **Features:** 
  - Overview of all ongoing private conversations.
  - Direct exchange with recruiters or specialists in real-time.

### 4. Contacts (Specialists)
- **Function:** A directory of all Würth experts on the platform.
- **Features:** 
  - View the profiles and expertise of the specialists.
  - Start a private chat (DM) instantly with the click of a button.

### 5. Career & Talent Pool
This section adapts based on the user's role:
- **For Students:**
  - **Job Offers:** List of all current jobs, internships, and working student positions.
  - **My Profile:** Students can add their skills, modules, grades, and availability here. This profile can be discovered by Würth recruiters.
- **For Admins / HR:**
  - **Job Management:** Create, edit, or delete jobs.
  - **Talent Pool Browser:** A search engine to filter students by specific skills (e.g., React, Python) or degree programs, and contact them directly if needed.

### 6. Admin Analytics (Admins Only)
- **Function:** An overview dashboard for management.
- **Features:** 
  - Live statistics on user numbers, messages, and talent pool size.
  - Activity history charts.
  - Search and manage the user list.

### 7. My Profile
- **Function:** Manage your own user data.
- **Features:** 
  - Upload a profile picture.
  - Select your university from a comprehensive list.
  - Update your biography and career status (e.g., Student, Intern).
  - Display collected certificates and status badges.
